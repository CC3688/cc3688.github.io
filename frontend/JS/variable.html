# 变量/赋值

# 变量

# 变量声明

var let const

# var

  • 支持变量声明预解析
  • 不支持块作用域
  • 允许重复声明

# let

  • 不支持变量声明预解析（先声明后使用）
  • 支持块作用域
  • 不允许重复声明（暂存死区）

# const

  • 不支持变量声明预解析（先声明后使用）
  • 支持块作用域
  • 不允许重复声明（暂存死区）
  • 常量（值一旦确定，不允许修改，所以必须初始化）

# 总之, var不要用了, 尽量都用 const, 当你需要变时才用 let

# 赋值

# 解构赋值

允许按照一定模式，从数组和对象中提取值，并对变量进行赋值，这被称为解构赋值

需要注意:

  • 1.左右两边必须一样；右边得是个东西(数组,对象...)
  • 2.必须定义和赋值同步完成

# 数组解构

const [a, b, c] = [1, 2, 3];  
// 顺序对应
// 即有了
// const a = 1
// const b = 2
// const c = 3

# 对象解构

const {foo, bar} = {foo: "aaa", bar: "cc3688"}
//key 值对应
// 即有了 
// const foo = "aaa"
// const bar = "cc3688"

# 解构赋值-别名

const {foo: f, bar: b} = {foo: "aaa", bar: "cc3688"}
//foo 原始名称
// f 别名
// const f = "aaa"  (不用产生 const foo  这个变显的)
// const b = "cc3688"

# 多重解构

let { foo: [a, b] } = { foo: [10,20], bar: "bbb" }
