# 数组

数组是极其常用的

# map

映射: 一个对一个

const arr1 = [18, 67, 98, 25, 17, 96] 
const arr2 = arr1.map(i=> i >= 60)

// [ 18, 67, 98, 25, 17, 96 ]
// [ false, true, true, false, false, true ]
// 映射, 一个对一个 弄一个新的出来,  原数组不变, 返回一个新的数组

# reduce

汇总：一堆 汇总(总之是经过一系列操作最后得到) 一个

const arr = [18, 67, 98, 25, 17, 96]
const res = arr1reduce((tmp, item, index)=> {
   
    if(index === arr.length - 1) {

// 汇总后, 返回一个结果

# filter

const arr = [18, 67, 98, 25, 17, 96] 
const res = arr.filter((value,key,arr)=> {
    
    return v > 60
})

// 回调函数有三个参数, 数组元素, 该元素的索引, 该数组   一般就用一个参数(第一个)
// 它必须 返回 布尔值, 当为真时, 该元素被保存到res中, 为假不保存
// 返回一个新的数组, 不改变原数
